### 개발기간 : 2023.07 ~ 2023.09
### 프로젝트 인원 : 3명
### 역할 : Oracle Cloud DB구축, 서버 개발, 앱 개발
### 상세 개발 : 
- Oracle Cloud DB를 통해 공유DB 구축
- Tomcat을 통해 웹 서버를 구축하고 java & jsp를 이용하여 android <-> DB 간 통신을 담당하는 기능 개발
- App(질환백과, 자주찾는 질병정보) 개발
### 개발 환경 및 언어 : 
- Visual Studio
- Flutter
- Eclipse
- Tomcat
- Oracle Cloud
- Dart
- Java
- JSP
